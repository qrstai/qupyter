Unittest Helpers#

qupyter 전략 단위 테스트를 위한 Mock 객체들을 정의합니다.

아래와 같은 방법으로 사용할 수 있습니다.

examples:

import asyncio
from qupyter import test
from qupyter.brokerage.ebest import EBestStockBroker

def trade_func(account_info, pending_orders, positions, stock_broker):
    # 사용자 전략 코드 작성
    # ...

if __name__ == '__main__':
    broker = EBestStockBroker(test_trade=True)
    account_info = broker.get_account()

    positions = [
        test.position(asset_code='005930', quantity=10, average_purchase_price=68000, current_price=70000),
        test.position(asset_code='252670', quantity=10, average_purchase_price=2800, current_price=3000),
    ]

    pending_orders = [
        test.order(asset_code='005930', trade_type='1', quantity=10, price=68000),
        test.order(asset_code='252670', trade_type='2', quantity=10, price=2800),
    ]

    result = trade_func(account_info, pending_orders, positions, broker)
    print(result)
qupyter.test.order(**kwargs) → StockOrder#

StockOrder Mock 객체 생성

아래 필드들은 모두 선택적으로 지정할 수 있습니다.

Parameters:

  • order_id (int) – 주문번호 (기본값: 1~999999 사이의 임의의 정수)

  • asset_code (str) – 종목코드 (기본값: ‘005930’)

  • trade_type (str) – 매매구분 (1: 매도, 2: 매수) (기본값: 1)

  • quantity (int) – 주문수량 (기본값: 1)

  • price (int) – 주문가격 (기본값: 68000)

  • filled_quantity (int) – 체결수량 (기본값: 0)

  • filled_price (int) – 체결가격 (기본값: 0)

  • pending_quantity (int) – 미체결수량 (기본값: quantity - filled_quantity)

  • order_time (datetime) – 주문시간 (기본값: 현재시간)

  • order_method (str) – 주문방식 (기본값: ‘TEST’)

  • current_price (int) – 현재가 (기본값: 68000)

Returns:

주문 객체

Return type:

StockOrder

qupyter.test.position(**kwargs) → StockPosition#

StockPosition Mock 객체 생성

아래 필드들은 모두 선택적으로 지정할 수 있습니다.

Parameters:

  • asset_code (str) – 종목코드 (기본값: ‘005930’)

  • asset_name (str) – 종목명 (기본값: ‘삼성전자’)

  • quantity (int) – 보유수량 (기본값: 1)

  • exit_available_quantity (int) – 청산가능수량 (기본값: quantity)

  • average_purchase_price (int) – 평균매입가 (기본값: 68000)

  • purchase_value (int) – 매입금액 (기본값: average_purchase_price * quantity)

  • loan_value (int) – 대출금액 (기본값: 0)

  • loan_date (datetime.date) – 대출일 (기본값: None)

  • expiration_date (datetime.date) – 만기일 (기본값: None)

  • current_price (int) – 현재가 (기본값: 68000)

  • commission (int) – 수수료 (기본값: 0)

  • tax (int) – 세금 (기본값: 0)

  • loan_interest (int) – 대출이자 (기본값: 0)

  • current_value (int) – 평가금액 (기본값: current_price * quantity)

  • current_pnl (int) – 평가손익 (기본값: (current_price - average_purchase_price) * quantity - commission - loan_interest)

  • current_pnl_pct (float) – 수익률 (기본값: current_pnl / (average_purchase_price * quantity) * 100)

Returns:

포지션 객체

Return type:

StockPosition
